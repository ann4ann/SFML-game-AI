# Product Context

## Project Overview
SFMLgame is a 2D space shooter developed in C++17 using the SFML 3.0 library.

## Project Goal
To create a fully functional 2D arcade space shooter game with classic mechanics: ship control, shooting enemies, scoring, levels, and bosses.

## Key Features (planned)
- Player controls (keyboard/gamepad)
- Enemies with different behavior patterns
- Scoring and high-score system
- Multiple difficulty levels
- Boss battles
- Visual effects (particles, animations)
- Sound (SFX + music)
- Menu and Game Over screen

## Target Audience
Fans of retro arcade games and space shooters. The game should run on Windows, Linux, and macOS.

## Current Status
Early stages. There is a basic CMake project with SFML 3.0 enabled and a minimal main.cpp that displays a window with a green background and a cyan rectangle.
